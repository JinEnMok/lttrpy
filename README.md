# lttrpy
A CLI Python script to compare users' Letterboxd "watched" lists, with ratings and stars

### Usage

```
python lttrpy [-h] [-v] user1 [user2 ...]
```

### Requirements

* Python >= 3.9
* `requests`
* `beautifulsoup4`
* Optional, for slightly faster execution:
  * `lxml`
  * `faust-cchardet`
